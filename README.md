# noah
noah will let you know the water level of the nearest river in the menu bar.

## To Use
```
# Clone this repository
git clone git@github.com:Noah0x0/noah.git
# Go into the repository
cd noah
# Install dependencies
yarn install
# Run the app
yarn start
```
